# id-knmi

Information Design Project KNMI

# Usage

## Clone Project

```
git clone <reponame>
```

```
cd id-knmi
```

## Install Project

### Server

```
npm install
```

### Client

```
cd client
```

```
npm install
```

## Run Project Locally

run this command from `/id-knmi`

**Start Server**

```
npm run start
```

**Start Client**

```
npm run dev
```
